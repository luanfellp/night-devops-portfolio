## 📌 Sobre o projeto

Projeto prático desenvolvido para estudo de arquitetura cloud e automação utilizando AWS.

O objetivo foi implementar um fluxo completo de deploy contínuo para uma aplicação estática, aplicando conceitos utilizados em ambientes reais.

## 🚀 Arquitetura

- AWS S3 (Static Hosting)
- CloudFront (CDN)
- GitHub Actions (CI/CD)

## ⚙️ Funcionalidades implementadas

- Deploy automatizado via pipeline
- Versionamento com Git
- Distribuição via CDN
- Estrutura simples e escalável

## 🎯 Objetivo

Consolidar conhecimentos em cloud computing, automação e boas práticas de DevOps em um cenário prático.
